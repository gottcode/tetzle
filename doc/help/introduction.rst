.. _introduction:

Introduction
************

Tetzle is a jigsaw puzzle game that uses tetrominoes for the pieces. Any
image can be imported and used to create puzzles with a wide range of
sizes. Games are saved automatically, and you can select between currently
in progress games.

-----------------
Starting the game
-----------------

You can start the game using the application menu of your desktop environment.
Open the application menu and choose :menuselection:`Games --> Tetzle` or open
a quick start prompt using :kbd:`Alt-F2` and start typing *tetzle*, then click
on the application name found.


-----------------
Menu bar overview
-----------------

As usual, all menu entries which have corresponding keyboard shortcuts, these
shortcuts are displayed directly next to the menu items.

:guilabel:`&Game` menu
----------------------

* :menuselection:`&Choose` – Starts a new game. For how to create a new game,
  see :ref:`start_new_game`.

* :menuselection:`&Retrieve Pieces` – Rearrange the pieces and reduce the
  spacing. The rotation of the pieces will be kept. This is helpful if you have
  zoomed in and some pieces have disappeared from the viewport.

* :menuselection:`&Quit` – Quits the game and the application.

:guilabel:`V&iew` menu
----------------------

* :menuselection:`Zoom &In` – Enlarge the view.

* :menuselection:`Zoom &Out` – Decrease the view.

* :menuselection:`Best &Fit` – Sets the zoom back to a level which fits in the
  window. If pieces have disappeared before, they come back.

* :menuselection:`Show O&verview` – Shows the picture what the puzzle is based
  on in a separate window. If you've lost track of what the puzzle picture
  looks like, you can refresh your memory here.

* :menuselection:`F&ullscreen` – Switches to fullscreen mode. In this mode, the
  :program:`Tetzle` window is displayed without the frame; you still have
  access to the menu bar and see the progress bar and the zoom slider at the
  bottom of the window.


:guilabel:`Se&ttings` menu
--------------------------

* :menuselection:`&Appearance` – Opens a new window where you can change the
  colors and some other aspects of the appearance.

* :menuselection:`&Language` – Lets you choose a different language for the
  user interface of :program:`Tetzle`.

See :ref:`configuration` about how to use the entries of the *Settings* menu.


:guilabel:`&Help` menu
----------------------

* :menuselection:`&Controls` – Shows a brief summary of the usage. In fact, the
  same as described in :ref:`howtoplay`.
